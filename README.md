# Go Balancer

A Vue 3 + Vite web application

## Project Structure

```
go-balancer-web/
├── src/
│   ├── components/        # Vue components
│   ├── composables/       # Shared composition logic
│   ├── services/          # API integrations
│   ├── types/            # TypeScript definitions
│   ├── utils/            # Helper functions
│   └── views/            # Page components
```

## Tech Stack

- Vue 3 + TypeScript
- Vite
- Tailwind CSS

## Setup

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Build for production
pnpm build
```

## Development

Built with Vue 3 `<script setup>` syntax and TypeScript. Uses composition API for state management and component logic.
